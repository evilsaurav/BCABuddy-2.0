## ğŸ” Common Issues & Solutions

### Export buttons disabled?
**Issue**: "Export as PDF" or "Export as CSV" greyed out
**Solution**: You need at least 1 message in chat history to export

### Settings not saving?
**Issue**: Settings reset after closing browser
**Solution**: 
- Check if browser allows localStorage
- Disable "Clear data on exit" in browser settings
- Try different browser

### Profile picture not uploading?
**Issue**: Upload button not working
**Solution**:
- File must be image format (PNG, JPG, WEBP)
- File size should be < 5MB
- Check internet connection

### Password change failed?
**Issue**: "Old password is incorrect" error
**Solution**:
- Verify you entered correct current password
- Check Caps Lock is off
- Password is case-sensitive
